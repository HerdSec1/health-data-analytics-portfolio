# Health Data Analytics Portfolio

This repository is a CHDA-aligned healthcare data analytics portfolio focused on turning healthcare data into decision-ready information.

It is built around the AHIMA Certified Health Data Analyst (CHDA) domains:

1. Foundational Knowledge of Analytics in Healthcare
2. Business Needs Assessment
3. Data Acquisition
4. Data Analysis
5. Data Interpretation and Reporting
6. Data Governance

## Purpose

The purpose of this portfolio is to demonstrate how healthcare data can be acquired, managed, analyzed, interpreted, reported, and governed in a way that supports operational decision-making.

This repository is not affiliated with any employer, healthcare organization, vendor, or electronic health record system.

## Data Disclaimer

All datasets, examples, scenarios, SQL queries, and reports in this repository are synthetic.

No protected health information (PHI), employer data, proprietary system information, internal workflows, or real patient data are included.

## Featured Project

### Project 01: Clinical Documentation Performance Analysis

This project uses a synthetic clinical documentation dataset to evaluate:

- Documentation status distribution
- Unsigned and pending note volume
- Documentation timeliness
- Department-level trends
- Provider workflow patterns
- Data quality and governance risks
